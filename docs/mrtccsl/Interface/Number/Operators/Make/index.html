Field includes all definitions and exioms of Ring, plus multiplicative inverse \forall a \neq 0: a\cdot a^{-1} = 1.

include Ring

Ring is assumed to satisfty the following axioms:

  • For addition:

    1. associativity (a + b) + c = a + (b + c)
    2. commutativity a + b = b + a
    3. 0-identity a + 0 = 0 + a = a
    4. existence of inverse a + (-a) = 0

  • For multiplication:

    1. associativity (a \cdot b) \cdot c = a \cdot (b \cdot c)
    2. 1-identity 1\cdot a = a\cdot 1 = a

  • For the two:

    1. left distributivity a\cdot (b + c) = (a\cdot b) + (a \cdot b)
    2. right distributivity (b+c) \cdot a = (b\cdot a) + (c \cdot a)
