CCSL+

This package contains libraries implementing MRTCCSL language and its Stochastic extension.

  • mrtccsl library: Mrtccsl
  • mrtccslparsing library: Mrtccslparsing

  • std library:

    • Prelude: it is intended to be the first opened; redefines some standard libraries with more functions
    • Boolean
    • Expr
    • Interfaces
    • Stats
    • Number
  • symbolic library provides Symbolic.Machine
